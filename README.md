# 🍽️ Spoonful Recipe Website

A modern and user-friendly recipe website that allows users to discover and explore recipes using the **Spoonacular API**. The project focuses on clean UI, usability, and efficient API usage by limiting the number of recipes displayed per request.

https://jenniferguinanzaca.github.io/RecipeProject/index.html

## ✨ Features

- 🥗 Browse recipes fetched from the Spoonacular API
- 🔍 Search recipes by name or ingredients
- 📖 View detailed recipe information
- ⏱️ Cooking time, ingredients, and instructions
- 📱 Fully responsive design
- ⚡ Optimized API usage by limiting results

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **API:** Spoonacular Recipe API  

## 🌐 API Usage (Important)

This project uses the **Spoonacular API**, which has a request and quota limit.

To stay within the API limits:
- The number of recipes displayed per request is **intentionally limited**
- Result caps are used to reduce excessive API calls

This ensures better performance and avoids exceeding the API’s daily quota.
